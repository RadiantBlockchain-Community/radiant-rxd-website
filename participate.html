    Overview

    Radiant's strength lies in its community. Every contribution, big or small, helps build a more robust blockchain ecosystem. Here's how you can get involved:

    Support Radiant

    Radiant thrives on active participation. By engaging with the network, you help strengthen its foundation and expand its capabilities. Here are key ways to support the Radiant Network:

    Use Radiant

    Integrate Radiant into your daily on-chain activities. Whether you're sending transactions, building, interacting with smart contracts, or exploring DApps, every action strengthens the network. Play with Photonic Wallet and Glyph Miner today.

    Be the Network

    Decentralization and activity is key to Radiant's success. Your participation, whether running a node or simply holding RXD, contributes to a more distributed and resilient network.

    Mine RXD

    Radiant was specifically designed to transition from GPU mining to ASIC mining for enhanced scalability and efficiency. The Radiant Network began with only GPU and FPGA miners available to mine with Proof of Work to earn RXD coins.

    Radiant now has ASIC miners publically available on the market as of September 5th 2024. The first manufacturer to announce a Radiant compatible SHA512/256d ASIC miner was DragonBall Miner, with the A11 Dual ASIC Miner.

    ICERIVER was the second manufacturer to release an ASIC miner for Radiant, with the ICERIVER RXD RX0.

    Buy Radiant RXD

    Acquiring and holding RXD coins is a fundamental way to support the Radiant ecosystem. Your participation in the coin and token economy helps drive growth and stability. More holders and Radiant ecosystem participants can strengthen the lindy effect. Here's why buying RXD matters:

    Increase RXD Liquidity

    A liquid market is crucial for any asset. By buying and holding RXD, you're contributing to market depth and stability. Exchanges allow for onboarding new users and participants to access the fuel that drives Radiant's ecosystem to thrive and grow.

    Grow the Userbase

    The more holders and users Radiant has, the stronger the network becomes. Your participation helps create a network effect, attracting more users and developers to the ecosystem.

    Run a Node

    Nodes are the backbone of the Radiant Network, ensuring its security, speed, and decentralization. By running a node, you become an integral part of Radiant's infrastructure. Consider running a Radiant Node and a public ElectrumX Server. Also learn more about the Radiant System Design and Radiant Node Types:

    Note: All nodes are full nodes until a future update is released, which will add Archive and Agent types to the network (only around 24GB for full nodes currently)

    Mining Node

    Beyond just mining, running a full Mining Node helps validate transactions and maintain the integrity of the blockchain.

    Agent Node

    These nodes will process and relay transactions, forming a nervous system of the Radiant Network. Running an agent node will help improve user experienced network speed and reliability.

    Archive Node

    By storing the full blockchain history, archive nodes will play a crucial role in maintaining the network's long-term integrity and providing data for analysis and re-propagation.

    Build on Radiant

    Developers are the architects of Radiant's future. By creating applications and tools on Radiant, you're expanding the ecosystem and demonstrating its potential.

    Radiant's infrastructure is designed for scalability and efficiency. Here are a few ideas that can help drive adoption and growth:

    • DeFi Protocols
    • Wallets
    • NFT Platforms
    • Decentralized Exchanges
    • Social DApps
    • Gaming Platforms
    • Or any other blockchain application

    Radiant provides the tools and support to bring your vision to life. Join our developer community and start building the next generation of blockchain applications.

    Current dev tooling available to use is:

    RadiantScript: A fork of CashScript with support for Radiant Opcodes. RadiantScript is a high-level language that allows you to write Radiant Smart Contracts in a straightforward and familiar way. Its syntax is inspired by Ethereum's Solidity language, but its functionality is different since the underlying systems have very different fundamentals.

    RadiantJS: A Radiant JavaScript Library

    RXD.WASM/RXD-RS: A Rust/WASM Library to interact with Radiant

    Glyph Protocol: Create Fungible Tokens, Non-Fungible Tokens, or Smart Contracts by utilizing digital signatures on Photons, the sub-units of Radiant's coin, RXD.

    Photonic Wallet: A non-custodial wallet for minting and transferring tokens, following the Glyphs Protocol. The code runs completely client side and only requires a connection to an ElectrumX server.

    Photonic Wallet - Extension: Utilize the Photonic Wallet extension available at the Chrome Web Store.

    Muon Pay SDK: A powerful and easy-to-integrate solution for merchants looking to accept payments on the Radiant (RXD) blockchain.

    Radiant Server Setup Guide: Easy to follow guide on how to setup and run a Radiant Node and ElectrumX Server.

    Glyphs Protocol Tech Guide: Learn the technical details on how the Glyphs Protocol works on Radiant.
